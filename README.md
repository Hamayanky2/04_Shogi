# はじめに
将棋についての知識や経験のまとめです。

## 目的
三つの力を鍛えること

### 一 「攻守を知る」
    定跡とその意味を知ることで、基礎を学びながら応用力も鍛えることができます。<br>
    まずは定跡の組み方とメリット・デメリットを知ることが基礎的な棋力につながります。

### 二 「表裏を考える」
    全ての手は利点と欠点の両方の側面を併せ持ちます。<br>
    一手一手の「表裏」を考えながら指すように意識します。

### 三 「相手を読む」
    手の先を読むだけではなく、その一手にどのような目的があるかを理解することが重要です。

## 目標
    アマでの段位獲得を目指します！

# はじめに
ここに記述されるのは将棋についての知識や経験のまとめです。

## 目的
三つの力を鍛えること

### 一 「攻守を知る」
定跡とその意味を知ることで、基礎を学びながら応用力も鍛えることができます。
まずは定跡の組み方とメリット・デメリットを知ることが重要です。

### 二 「表裏を考える」
全ての手は利点と欠点の両方の側面を併せ持ちます。
一手一手の「表裏」を常に考えて指すことが重要です。

### 三 「相手を読む」
未来を見ることができるからといって勝利をつかめるわけではありません。
先を読むだけではなく、相手の一手の意図を理解することが重要です。
